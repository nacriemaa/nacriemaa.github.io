# nacriemaa.github.io

## Steps to run

### Run locally
```bash
export GEM_HOME="$HOME/.gem"
gem install bundler -v 2.4.22
bundle instal
bundle exec jekyll serve --config _config.yml,_config_dev.yml --watch
```
